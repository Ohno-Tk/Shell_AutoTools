# タイムスタンプ変更ツール
![画面](Readme__image\image1.JPG)


## 使い方
1. run.batを実行。
1. 変更箇所にチェックを付ける。
1. 変更したい時刻を入力。
1. 実行ボタンを押下。
1. 変更したいファイルを選択。
